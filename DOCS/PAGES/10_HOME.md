# Editing the Home Page.

## Add a new Homepage Resource Tile:
- Go to Manage >> Content >> Add Content >> Homepage Tile
- Title is what the resource's name is
- Image and Summary should be provided to you
- Link can either be added as a Tile Link or link-alt
- Use Title Link if it is a link within RCD
- Use link-alt and use HTML to make the link pop-out if it links to external (see 'Artificial Intelligence' Homepage Tile)

## Edit/Delete Homepage Resource Tile:
- Go to Manage >> Content
- Use the 'Content Type' dropdown and change it to 'Homepage Tile'.  

## Edit Community Content Type / Node:
- Structure >> Content Types >> Homepage Tile

## Edit the Communities View:
- Structure >> Views >> Homepage Resource Tiles	
- The view is mostly structured by custom templating and css.
